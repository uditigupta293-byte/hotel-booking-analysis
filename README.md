# Hotel Booking Data Analysis 📊

A complete exploratory data analysis project on hotel booking behavior using Python and Google Colab.  
This project focuses on booking trends, cancellation patterns, customer behavior, and business insights extracted from hotel reservation data.

---

## Project Objective 🎯

The goal of this project is to analyze hotel booking data and answer key business questions such as:

- Why do customers cancel bookings?
- Which months receive the highest bookings?
- How does lead time affect cancellations?
- What differences exist between hotel types?
- What customer patterns can hotels use to improve revenue?

---

## Dataset Overview 🗂️

The dataset contains hotel reservation records with features such as:

- Hotel Type
- Booking Cancellation Status
- Lead Time
- Arrival Date
- Stay Duration
- Adults / Children / Babies
- Meal Plan
- Country
- Market Segment
- Customer Type
- ADR (Average Daily Rate)

## Major Insights 🔍

- Higher lead time bookings tend to cancel more frequently.
- Certain months show peak reservation demand.
- City Hotels and Resort Hotels show different customer behavior.
- ADR varies based on season and booking type.
- Repeated guests behave differently than new customers.

---

## Business Recommendations 💡

- Offer early confirmation incentives for long lead-time bookings.
- Increase pricing during peak months.
- Target repeat customers with loyalty benefits.
- Improve cancellation policies strategically.
- Run seasonal promotions during low-demand months.
